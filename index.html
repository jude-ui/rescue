# 모바일 홈화면에 바로가기 아이콘 만들기

차에서 위치를 놓치지 않도록 홈 화면에 아이콘을 추가해 보세요.

- IOS 사파리 하단 공유하기 버튼 > 홈 화면에 추가 - 안드로이드 삼성인터넷 우측 하단 더보기 메뉴 > 현재 페이지에 추가 > 홈 화면 - 네이버 앱 우측 하단 더보기 메뉴 > 홈 화면에 추가 - 크롬 브라우저 우측 상단 더보기 메뉴(... 아이콘) > 홈 화면에 추가 # ‘홈 화면에 추가’ 항목이 안보일 경우

아래 버튼을 클릭하여 주소를 복사한 뒤, IOS의 사파리 또는 삼성인터넷에 주소를 붙여넣어서 위에 설명된 방법으로 홈 화면에 추가해 주세요.
